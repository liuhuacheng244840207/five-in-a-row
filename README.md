# 五子棋游戏 (five-in-a-row)

## 一、项目介绍
这是一个基于 C 语言开发的控制台版五子棋游戏，支持双人对战模式，遵循标准五子棋规则（率先在棋盘上形成五子连珠的一方获胜）。

## 二、关键文件

1.  **项目入口：五子棋.sln**
    这是 Visual Studio 的解决方案文件，双击即可在 Visual Studio中打开整个项目
2.  **编译结果：五子棋.exe**
    项目编译后生成的可执行文件，路径：x64->release->五子棋.exe
    直接双击即可运行五子棋游戏（无需打开 Visual Studio）。

## 三、游戏操作说明
- 启动游戏后，在主菜单输入 `1` 开始游戏，`3` 退出游戏；
- 游戏界面中，棋盘坐标范围为 `0~18`（x 轴：列，y 轴：行）；
- 落子时输入格式为 `x y`（如 `5 8` 表示在第 5 列、第 8 行落子）；
- 黑棋（●）先落子，黑白棋交替行棋，率先形成五子连珠者获胜；
- 获胜后按回车键返回主菜单。

## 四、核心逻辑说明
- **棋盘存储**：使用 `int board[19][19]` 二维数组，`0` = 空、`1` = 黑棋、`2` = 白棋；
- **胜负判定**：`isWin()` 函数通过方向数组检测落子位置的 4 个方向（横 / 竖 / 右下 / 左下）是否形成五子连珠；
- **落子校验**：`playerMove()` 函数校验落子位置是否为空，确保落子合法性。
